# Wind and Snow Loads Calculator (Eurocode 1)
## roject Structure
* `main.py` - The main execution script.
* `anemosCALC.py` - Core logic and formulas for wind load calculations.
* `anemosData.py` - Contains the necessary data tables, parameters, and constants for wind actions.
* `snow.py` - Core logic and formulas for snow load calculations.
* Disclaimer
This tool is intended for educational and preliminary design purposes. Always verify calculations with certified structural engineering software and the official Eurocode documents.
